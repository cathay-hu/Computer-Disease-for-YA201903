<h2>使用說明
<p>1.下載BD.exe，儲存在隱蔽位置
<p>2.在與BD.exe同一目錄下即同一文件夾創建文本文檔"time.txt",依次輸入設定的年、月、日、時、分、秒，用空格隔開，保存。
<p>3.打開BD.exe
<p>解除：發作時:重啟電腦;發作前或重啟後：打開程序旁邊的"data.txt"，把裡面的數字修改成1，再運行BD.exe
